Sample Task for ACT

## Target

See [Details](task_details.pdf)

## Running
You need to have Flutter SDK in your computer, with it's executable path (bin folder) added in your terminal's path
Then,
1. `flutter pub get`
2. `flutter run`

This app can be run on Android/iOS/Web
